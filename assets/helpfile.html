Life Vault Help

  • What is Life Vault?

  • The Life Vault app allows the user to store personal entries on their phone, recording information about some event, observation, or thought which can later be retrieved, edited, and shared.

  • What are the features?

    • List of entries
    • Create, view, edit and delete entry
    • Sort list of entries. Save previous sort setting using Shared Preferences
    • Duplicate an entry
    • Location and Orientation of an Entry
    • Multiple photos
    • Multiple video recordings
    • Multiple voice recordings
    • Send an entry via Email using html
    • Links to relevant media
    • Links to relevant contacts
    • Display help information, using html and WebView

  • How to create a new entry

  • Press the little plus button on the top left on the starting screen. Afterwards you see a new empty entry. By clicking on the Title or Text field you can enter what you want. Date is set automatically, each time you make a change.

  • How to edit an entry?

  • Choose the entry you want to edit from the list. By clicking on each element within the entry you can edit it.

  • How to set a main picture?

  • To set a main picture for each entry, perform a long click on any picture, previously added to this entry. Choose the option "set as main picture"

  • How to delete an entry?

  • Perform a longclick in the list view on an item. Select "delete" from the dialog box.

  • How to duplicate an entry?

  • Perform a longclick in the list view on an item. Select "duplicate" from the dialog box.

  • How to send an entry via Email?

  • Perform a longclick in the list view on an item. Select "send via email" from the dialog box.
